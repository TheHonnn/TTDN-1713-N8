================================================================================
🔌 API ENDPOINTS
================================================================================

1. POST /api/face_attendance/checkin
   Nhận dữ liệu chấm công từ AI Service
   Headers: X-API-Key: your_api_key
   ✓ Auto-create/update attendance
   ✓ Auto-detect check-in vs check-out
   ✓ Detailed logging

2. GET /api/face_attendance/employee/<id>
   Lấy thông tin nhân viên
   ✓ Kiểm tra face_registered

3. GET /api/face_attendance/logs?limit=10
   Lấy danh sách logs
   ✓ Filter by employee_id, status

4. GET /api/face_attendance/health
   Health check endpoint
   ✓ Không cần authentication

📊 DATABASE SCHEMA
================================================================================

Table: face.attendance.log
├─ employee_id (Many2one: hr.employee)
├─ department_id (Related: hr.department)
├─ check_time (Datetime)
├─ check_type (Selection: check_in, check_out)
├─ confidence_score (Float: 0-1)
├─ distance_score (Float)
├─ recognition_status (Selection: success, low_confidence, no_match, error)
├─ camera_source (Char)
├─ image_snapshot (Image)
├─ attendance_id (Many2one: hr.attendance)
├─ is_synced_to_attendance (Boolean)
├─ api_request_data (Text: JSON)
├─ api_response_data (Text: JSON)
├─ request_source_ip (Char)
├─ is_flagged (Boolean)
├─ flag_reason (Selection)
├─ notes (Text)
├─ created_at (Datetime)
└─ processed_at (Datetime)

Extended hrEmployee:
├─ face_encoding (Text: JSON array)
├─ face_image (Image)
├─ is_face_registered (Boolean: computed)
└─ face_confidence_threshold (Float: default 0.55)

📞 TROUBLESHOOTING
================================================================================

❌ "ImportError: No module named 'cv2'"
  → pip install opencv-python

❌ "Cannot open webcam"
  → ls /dev/video*
  → sudo usermod -a -G video $USER

❌ "API connection failed"
  → Check Odoo is running: ps aux | grep odoo
  → Check port: lsof -i :8069

❌ "Employee not found"
  → Use correct employee ID from Odoo HR

❌ "Low confidence match"
  → Improve lighting, get closer, look at camera directly

Chi tiết: Xem INSTALLATION_GUIDE.md → Troubleshooting section
